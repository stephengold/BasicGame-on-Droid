# BasicGame-on-Droid

jMonkeyEngine's prototypical `BasicGame` as an Android/Java project
<img height="150" src="https://raw.githubusercontent.com/stephengold/BasicGame-on-Droid/refs/heads/master/screenshot.png" alt="BasicGame screenshot">

If you want to get started using jMonkeyEngine
with Android SDK 2025.1.3 or higher, use this project as a template.

Or you could simply fill out a web form:
[the JMonkeyEngine Initializer](https://jmonkeyengine.org/start/).
